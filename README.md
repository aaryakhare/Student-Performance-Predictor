# 🎓 Student Performance Predictor

## 📌 Project Overview

The Student Performance Predictor is a Machine Learning project that predicts a student's exam score based on the number of hours they study. The project uses Linear Regression from Scikit-learn and demonstrates the complete machine learning workflow from data exploration to prediction.

---

## 🚀 Features

- Load student data from a CSV file
- Explore and analyze the dataset
- Visualize data using scatter plots
- Train a Linear Regression model
- Evaluate model performance using the R² Score
- Save the trained model using Joblib
- Predict exam scores for new study hours

## ▶️ How to Run

### Install dependencies

```bash
pip install -r requirements.txt
```

### Explore the dataset

```bash
python src/data_exploration.py
```

### Train the model

```bash
python src/train_model.py
```

### Make predictions

```bash
python src/predict.py
```
